Sample app made by Hudo Assenco and Nelson Dornelas for CS166 class.

This app uses an authentication system based on Tokens. The backend is made with NodeJS and the database with MongoDB. The Tokens are generated following the JSON Web Token standard implemented by the library node-jsonwebtoken.

The certificate is signed by Let's Encrypt. The server is hosted on an Amazon EC2 Instance running the Nginx web server.

The client is implemented using the beta version of Angularjs2 only for learning purposes. The token is saved in the localStorage of the broswer and later used in every request to the server to authenticate the user, making the server stateless.

Security aspects:
  • HTTPS makes the connection more secure, preventing a man in the middle attack.
  • Tokens allows a stateless servers, that are more efficient and scalable
  • Tokens are not stored in cookies, preventing CSRF attacks. On the other hand, storing tokens in the local storage of the browser makes it vulnerable to XSS attacks.
